# ReminderBot — Контекст проєкту

## Опис проєкту
Telegram-бот для створення, керування та відстеження нагадувань з підтримкою інтервалів, разових та повторюваних завдань, збереженням у SQLite (aiosqlite) та розкладом через APScheduler.

## Поточний стан
- **Поточна гілка Git:** `main` (успішно закомічено та запушено на GitHub!)
- **Основні бібліотеки:** `python-telegram-bot==22.5`, `APScheduler==3.11.0`, `aiosqlite==0.19.0`
- **Реалізовано:** 
  1. **Single-Message Inline Wizard з кнопкою ◀️ Назад:** Повний покроковий процес створення нагадувань відбувається в **одному повідомленні** з можливістю редагування кроків і вводу довільного часу/інтервалів з клавіатури!
  2. **Інтерактивний Дашборд з кнопками `✅ Готово` та `⏸ Відкласти`:** Кнопки прямо у картках нагадувань відмічають/відкладають саме найближче майбутнє нагадування (з урахуванням мульті-часових одноразових нагадувань!).
  3. **Авто-видалення одноразових завдань:** Мульті-часові одноразові завдання автоматично видаляються з БД, коли виконано всі їхні часові слоти.
  4. **Лаконічне Головне Меню (2 кнопки):** Прибрано застарілі кнопки.
  5. **Аудит Git:** Видалено `reminders.db` та `.pyc` з Git-відстеження, оновлено `.gitignore`.

## Що зроблено (Останні зміни)
- **`bot/ui_helpers.py`:**
  - Додано підтримку кнопок `◀️ Назад` у клавіатурах майстра та відображення кастомних інтервалів.
  - Виправлено візуальне відображення галочок днів для одноразових завдань.
- **`bot/keyboards.py`:**
  - Спрощено `MAIN_KEYBOARD` до 2 кнопок (`➕ Створити нагадування`, `📋 Мої нагадування`).
  - Додано `✅ Готово` та `⏸ Відкласти` у `build_dashboard_keyboard`.
- **`core/scheduler.py`:**
  - Додано метод `get_next_reminder_instance(user_id, task_id)` для визначення найближчого майбутнього нагадування.
- **`bot/handlers.py`:**
  - Додано обробку `dashdone_` та `dashsnooze_` для дашборду.
  - Очищено застарілі обробники.
