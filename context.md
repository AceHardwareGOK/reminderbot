# ReminderBot — Контекст проєкту

## Опис проєкту
Telegram-бот для створення, керування та відстеження нагадувань з підтримкою інтервалів, разових та повторюваних завдань, збереженням у SQLite (aiosqlite) та розкладом через APScheduler.

## Поточний стан
- **Поточна гілка Git:** `feature/rich-ui`
- **Основні бібліотеки:** `python-telegram-bot==22.5`, `APScheduler==3.11.0`, `aiosqlite==0.19.0`
- **Реалізовано:** Повний редизайн інтерфейсу на Telegram Bot API MarkdownV2 з безпечним екрануванням, Rich Visual Cards, майстром створення з прогрес-баром та інтерактивною дашборд-пагінацією.

## Що зроблено (Останні зміни)
- **Новий модуль `bot/ui_helpers.py`:** 
  - Реалізовано функцію `escape_md` для захисту від крашів при вводи спецсимволів (`*`, `_`, `(`, `)` тощо).
  - Реалізовано форматування карточок задач `format_task_card`, сповіщень `format_reminder_notification` та прогрес-барів `format_progress_header`.
- **Інтерактивна пагінація (`bot/keyboards.py` & `bot/handlers.py`):**
  - Замість вивантаження спаму окремих повідомлень для кожного нагадування створено інтерактивний Dashboard з кнопками `◀️`, `▶️` та швидкими діями.
- **Оновлення на `MarkdownV2`:**
  - Оновлено `bot/handlers.py`, `bot/edit_handlers.py` та `core/scheduler.py`.

## Наступні кроки
- Провести тестування в бота в реальному чаті Telegram.
- У разі успішного тестування — злити гілку `feature/rich-ui` у `main` (`git checkout main && git merge feature/rich-ui`).
